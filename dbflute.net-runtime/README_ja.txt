
# ========================================================================================
#                                                                                 Overview
#                                                                                 ========
DBFlute.NETランタイムのライブラリプロジェクト。
DBFlute.NETモジュールによって自動生成されたクラスが参照するクラスなどが含まれる。
DLLとして提供される。


# ========================================================================================
#                                                                              Environment
#                                                                              ===========
# ----------------------------------------------------------
#                                                    Compile
#                                                    -------
M2EなどのMaven管理ができる環境が整っていればビルドすることができます。


# ----------------------------------------------------------
#                                                  Namespace
#                                                  ---------
内部利用のJavaライクなクラスのための Namespace に加えて、
あとは、Java版のDBFluteランタイムと同じ。

DBFlute
 |-JavaLike
 |  |-Extensions
 |  |-Collections
 |  |-JDBC
 |
 |-(その他、Javaと同じ)
